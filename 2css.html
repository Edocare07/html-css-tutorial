Modalità di inserimento dei CSS

Modalità 1: css in linea

Il CSS viene quindi incorporato all'interno della pagina utilizzando l'elemento < style > < /style >



















Modalità 2: Css con foglio esterno collegato alla pagina html

Con un foglio di stile esterno, puoi cambiare l'aspetto di un intero sito web cambiando solo un file! Ogni pagina HTML deve includere un riferimento al file del foglio di stile esterno all'interno dell'elemento < link >, all'interno della sezione head.





















Modalità 3: css interno ad html nell'head

Un foglio di stile interno può essere utilizzato se una singola pagina HTML ha uno stile unico. Lo stile interno è definito all'interno dell'elemento < style >, all'interno della sezione head.
