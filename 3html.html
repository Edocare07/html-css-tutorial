Link e immagini

I link

I collegamenti vengono creati utilizzando l'elemento < a >. Gli utenti possono fare clic su qualsiasi cosa tra il tag di apertura < a > e il tag di chiusura < /a >. Tu specifichi a quale pagina desideri collegarti utilizzando l'attributo href. Il testo tra l'apertura tag < a > e tag di chiusura < /a > è noto come testo di collegamento. Dove possibile, il testo del collegamento dovrebbe spiegare dove saranno i visitatori preso se ci cliccano sopra (piuttosto piuttosto che dire "clicca qui"). Di seguito potete vedere il link a IMDB. Molte persone navigano nei siti web scansionando il testo per i collegamenti. Il testo del link chiaro può aiutare i visitatori trova quello che vogliono. Questo darà loro un aspetto più positivo impressione del tuo sito e maggio incoraggiarli a visitarlo per più a lungo. (Aiuta anche le persone utilizzando il software di lettura dello schermo.) Per scrivere un buon testo di collegamento, puoi pensa alle parole che le persone potrebbero utilizzare durante la ricerca di pagina a cui ti stai collegando. (Ad esempio, invece di scrivere "posti in cui soggiornare" potresti usare qualcosa di più specifico come "hotel a New York.")












Differenza link relativi e link assoluti:

Link assoluto: Ogni web la pagina ha il proprio URL. Questo è il indirizzo web che vorresti digitare in un browser se lo desideri visita quella pagina specifica. Un URL assoluto inizia con il nome di dominio di quel sito, e può essere seguito dal sentiero a una pagina specifica. Se nessuna pagina lo è specificato, il sito visualizzerà il homepage.

























Link relativo: Quando ci si collega ad altre pagine all'interno dello stesso sito, puoi utilizzare URL relativi. Oppure in altre parole possiamo dice che un link relativo sono percorsi che fanno riferimento alla posizione degli altri file rispetto al documento in cui ci si trova in quel momento. Le immagini:

Per aggiungere un'immagine in una pagina Html utilizzo il tag < img > nella sezione body del documento. E' usato per visualizzare immagini in formato gif, jpg o png nel documento ipertestuale. L'indirizzo url del file immagine va indicato nell'attributo SRC. Può essere un indirizzo url assoluto o relativo. Quello scritto dentro ad alt="...", compare nel caso il browser ha problemi nel caricamento e quindi compare quella scritta per avvisarti cosa ci sarebbe stato con qualche parola.
Spesso un elemento < img > può contenere altri due attributi che specificano le dimensioni dell'immagine
- Height: specifica l'altezza dell'immagine in pixel
- Widh: specifica la largezza dell' immagine in pixel
Negli ultimi anni però le dimensioni delle immagini sono controllate dal css e non da html.
